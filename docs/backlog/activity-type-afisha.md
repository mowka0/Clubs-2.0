# Активность типа «Афиша»

**Статус:** open · **Создано:** 2026-05-23 · **Origin:** brainstorm на этапе унификации создания активностей (variant 2 — ActionSheet picker)

## Идея

Третий тип активности (после `event` и `skladchina`) — **«Афиша»**: лёгкая публикация-анонс без RSVP-механики, без сбора денег, без лимита участников.

Зачем это отдельный тип, а не «событие без участников»:
- семантика разная: афиша — это «обратите внимание / приходите / посмотрите», без подсчёта явок
- UI карточки другой: фокус на обложке/тексте, а не на «12/20 идут»
- не требует attendance-флоу, не требует stage_1/stage_2 lifecycle

## Сценарии

- «На следующей неделе мастер-класс от внешнего гостя — приходите кто хочет»
- «Опубликовали статью / видео — гляньте»
- «Наш клуб упоминают в новостях»
- «Распродажа мерча»
- Любой контент-анонс, который не требует RSVP/денег

## Минимальный набор полей

- title (обязательно)
- description / body (rich text? markdown?)
- coverImage (опционально, но именно картинка делает афишу «афишей»)
- optional CTA: внешняя ссылка + label кнопки (например «Купить билет», «Читать»)
- publishedAt / expiresAt (опционально — самоудаление из ленты через N дней)

## Где появляется

- В Manage у организатора — в picker'е `+ Создать` третьим пунктом
- В общей ленте «Активности» (и в Manage, и в ClubPage для участников) — отдельный визуальный стиль карточки: обложка на всю ширину + заголовок + CTA
- Возможно — в discovery-ленте (PRD §«афиша активностей города» — это уже про дискавери). Тут отдельный разговор.

## Открытые вопросы

- Moderation: афиша = свободный текст, не структурированные поля. Нужна ли пре-модерация?
- Срок жизни: вечно в ленте? Авто-архив через 30 дней?
- Реакции / комментарии участников — отдельная история
- Привязка к существующему событию/сбору (афиша как «обложка-анонс» к event'у)

## Не делать пока

В текущей итерации унификации создания (variant 2) — только `event` + `skladchina`. Picker должен расширяться без редизайна, но третий пункт добавляется отдельной фичей, когда будет ясность по полям и moderation.
