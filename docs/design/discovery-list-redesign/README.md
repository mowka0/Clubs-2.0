# Редизайн списка клубов на главном экране (Discovery)

**Статус: exploration, ничего не залочено.** PO не нравится текущая лента больших
карточек-плиток (обложка 96px + тройка метрик, ~2,5 клуба на экран). Ниже 5
альтернатив зоны списка; шапка (аватар, hero, поиск, чипы) во всех мокапах
одинаковая — сравнивается только список.

Токены — актуальный Banco-Plata (`frontend/src/styles/brand-theme.css`), тёмная тема.
Мокапы: `mockups/*.html` (общий CSS — `mockups/_shared.css`).

## Варианты

| # | Файл | Идея | Плотность | Компромисс |
|---|------|------|-----------|------------|
| 1 | `01-compact-rows.html` | Плотный список-реестр: аватар 46px, имя, meta-строка, цена справа. Клуб со встречей сегодня — тёплая подсветка строки. | ~6–7 на экран | Обложки исчезают — визуальная айдентика клуба только цветом аватара |
| 2 | `02-grid-tiles.html` | Сетка 2 колонки: узкая полоса обложки (58px) + аватар внахлёст + имя в 2 строки. | ~4–6 на экран | Имена длинных клубов клампятся; метрики влезают только одной строкой |
| 3 | `03-ledger-index.html` | «Банковский реестр»: чистая типографика, порядковый номер, категорийный тик слева, табличные цифры. Самый «Banco-Plata» по духу. | ~7 на экран | Совсем без картинок — категории различаются только тиком и словом |
| 4 | `04-shelves.html` | Горизонтальные полки по смыслу: «Встречаются сегодня», «Самые живые», «Новые» + кнопка «Показать все». | 3 разреза каталога на экран | Полный каталог за отдельным кликом; полкам нужен минимум 8–10 клубов в городе, иначе пусто |
| 5 | `05-spotlight.html` | Гибрид: одна большая «карта дня» (лучший матч, с чипами-причинами) + остальные ультраплотными строками с категорийной точкой. | 1 «вау» + ~6 строк | Нужна логика выбора «клуба дня» на бэке (пока нет) |

## Заметки

- Все данные в мокапах фейковые, но структура полей = реальный `ClubListItemDto`
  + `ClubCardFactsDto` (имя, город, категория, участники, цена, вовлечённость,
  ближайшая встреча, ★ Топ-5).
- Вариант 4 и «клуб дня» в варианте 5 требуют новых разрезов API
  (полки/рекомендация); варианты 1–3 ложатся на существующие эндпоинты как есть.
- Скрытый ранг L3 (`★ Топ-5 в категории`) сохранён во всех вариантах, где есть
  место: пилюля (1), строка-★ (3, 5), текст в полке (4).
