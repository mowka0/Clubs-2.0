Поиск + фильтры — одна форма (4 механики)

Идея PO: объединить цену и категорию с панелью поиска. Механики кликабельны — жми лупу/строку. Кнопка — тема.
1Лупа-кнопка, поиск по тапу
Покой: строка = [🔍] + «Категория» + «Цена» — фильтры живут НА МЕСТЕ поиска. Тап по лупе — поле разворачивается на всю строку (фильтры уступают), ✕ — обратно. Экономия: 2 строки → 1.

2Аккордеон из строки поиска
Строка поиска всегда на месте; тап по ней (или по стрелке) раскрывает под ней панель: чипы категорий + пресеты цены. Выбрал — свернул. Один блок-«форма», как ты описал.

3Пилюли внутри поля
Фильтры — мини-пилюли ВНУТРИ поискового поля, справа (как теги в умных полях). Тап по пилюле — шторка. Одна строка всегда; текст плейсхолдера ужимается.

4Ужатый поиск + фильтры рядом
Буквально твоя вторая идея: поле поиска ужато (~половина), на освободившееся место — два дропа. Всё видно всегда, без раскрытий. Минус: плейсхолдер почти исчезает.
